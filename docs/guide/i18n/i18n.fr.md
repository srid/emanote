# Français

Cette page définit explicitement `page.lang: fr` dans sa
[[yaml-config|configuration YAML]]. Elle montre le chrome de
[[html-template|modèle]] par défaut d'Emanote en français.

Autres langues :

- [[guide/i18n|English]]
- [[i18n/i18n.zh|中文]]

## Ce qui change

Emanote utilise `page.lang` pour choisir les chaînes affichées par les modèles
HTML par défaut. Le pied de page, le titre de la [[toc|table des matières]], le
champ de [[search|recherche]], les boutons de copie et les pages d'index
intégrées utilisent la langue sélectionnée.

La plupart des sites devraient définir `page.lang` une seule fois dans leur
fichier [[yaml-config|`index.yaml`]] racine. Un réglage par page est surtout
utile pour une démonstration comme celle-ci.

```yaml
page:
  lang: fr
```

## Bouton de copie

Le titre du bouton de copie sur ce bloc devrait être affiché en français.

```haskell
main = putStrLn "bonjour"
```
