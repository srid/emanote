# Internationalisation en français

Cette page définit explicitement `page.lang: fr`. Elle montre le chrome par
défaut d'Emanote en français.

Pour la même note en anglais, ouvrez [[i18n.en|la démonstration anglaise]].

## Ce qui change

Emanote utilise `page.lang` pour choisir les chaînes affichées par les modèles
HTML par défaut. Le pied de page, le titre de la table des matières, le champ
de recherche, les boutons de copie et les pages d'index intégrées utilisent la
langue sélectionnée.

La plupart des sites devraient définir `page.lang` une seule fois dans leur
fichier `index.yaml` racine. Un réglage par page est surtout utile pour une
démonstration comme celle-ci.

```yaml
page:
  lang: fr
```

## Bouton de copie
