# Internationalisation in English

This page explicitly sets `page.lang: en`. It demonstrates the default English
template chrome used by Emanote.

For the same note written in French, open [[i18n.fr|the French demo]].

## What changes

Emanote uses `page.lang` to choose the strings rendered by the default HTML
templates. The footer, table-of-contents heading, search placeholder, copy
button labels, and built-in index pages all use the selected language.

Most sites should set `page.lang` once in their root `index.yaml`. Per-page
language settings are useful for documentation demos like this one.

```yaml
page:
  lang: en
```
