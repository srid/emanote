# Internationalisation

This page explicitly sets `page.lang: en` in its [[yaml-config|YAML configuration]].
It demonstrates the default English [[html-template|template]] chrome used by
Emanote.

Demos in other languages:

- [[i18n/i18n.fr|Français]]
- [[i18n/i18n.zh|中文]]

## What changes

Emanote uses `page.lang` to choose the strings rendered by the default HTML
templates. The footer, [[toc|table-of-contents]] heading,
[[search|search]] placeholder, copy button labels, and built-in index pages all
use the selected language.

Most sites should set `page.lang` once in their root [[yaml-config|`index.yaml`]].
Per-page language settings are useful for documentation demos like this one.

```yaml
page:
  lang: en
```
