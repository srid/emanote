# Developing Emanote

Expect humans to use /do to start work.

## Dependencies

Emanote depends on various dependencies (flake.nix inputs) that can be cloned in ~/code - for agent to work on and open PR (following the same /do workflow in that PR too). And then that PR can be used in Emanote by adding it to its flake.nix inputs.
